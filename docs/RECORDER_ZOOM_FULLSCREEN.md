# 錄影頁面全螢幕與縮放功能說明

## 🎯 新增功能

錄影頁面現已支援完整的檢視控制功能，讓您可以更清晰地查看攝影機畫面細節。

## ✨ 功能列表

### 1. **🔍 縮放模式（2x 放大）**
- **啟動方式：**
  - 點擊影像畫面任意位置
  - 點擊右上角 🔍 按鈕
  - 按鍵盤 `Z` 鍵
- **功能：**
  - 畫面放大 2 倍
  - 可拖動平移查看不同區域
  - 左上角顯示「2.0x」縮放指示
- **退出：**
  - 再次點擊畫面或 🔍 按鈕
  - 按 `ESC` 鍵

### 2. **⛶ 全螢幕模式**
- **啟動方式：**
  - 點擊右上角 ⛶ 按鈕
  - 按鍵盤 `F` 鍵
- **功能：**
  - 畫面全螢幕顯示
  - 支援滑鼠滾輪縮放（1x - 5x）
  - 即時顯示當前縮放倍率
- **退出：**
  - 點擊 ⛶ 按鈕
  - 按 `ESC` 鍵

### 3. **🎡 滾輪縮放（全螢幕專用）**
- **使用方式：**
  - 進入全螢幕模式
  - 使用滑鼠滾輪向上/向下滾動
- **功能：**
  - 縮放範圍：1.0x - 5.0x
  - 根據滑鼠位置智慧縮放
  - 即時顯示縮放倍率

### 4. **🖱️ 拖動平移**
- **使用方式：**
  - 在縮放模式下，直接拖動畫面
- **功能：**
  - 查看放大畫面的不同區域
  - 流暢的拖動體驗
  - 滑鼠游標自動變為抓手圖示

## ⌨️ 鍵盤快捷鍵

| 按鍵 | 功能 | 說明 |
|------|------|------|
| `F` | 切換全螢幕 | 針對第一個鏡頭 |
| `Z` | 切換縮放 | 針對第一個鏡頭 |
| `ESC` | 退出 | 退出全螢幕或縮放模式 |

## 🎮 操作流程

### 基本檢視流程
1. 開啟錄影頁面
2. 點擊「全部開始預覽」
3. 將滑鼠移到任一鏡頭畫面
4. 右上角出現控制按鈕
5. 選擇需要的檢視模式

### 詳細檢查對焦
```
1. 啟動預覽
2. 點擊畫面進入 2x 縮放
3. 拖動查看產品邊緣
4. 調整對焦滑桿
5. 觀察清晰度變化
```

### 全屏錄影監控
```
1. 啟動預覽
2. 點擊 ⛶ 進入全螢幕
3. 使用滾輪調整縮放倍率
4. 開始錄影
5. 按 ESC 退出全螢幕
```

## 💡 使用技巧

### 對焦調整最佳實踐
1. **使用 2x 縮放檢查細節**
   - 進入縮放模式
   - 查看產品邊緣是否銳利
   - 微調對焦滑桿
   
2. **全螢幕精確調整**
   - 進入全螢幕模式
   - 滾輪放大到 3x-4x
   - 檢查最細微的瑕疵
   
3. **快速比較兩個鏡頭**
   - 不進入全螢幕
   - 使用 2x 縮放模式
   - 快速切換查看

### 錄影監控建議
- 使用全螢幕監控單一鏡頭
- 長時間錄製時使用標準視圖
- 發現異常時立即放大檢查

## 🎨 視覺指示

### 縮放指示器
- **位置：** 左上角
- **顯示：** 
  - 標準模式：不顯示
  - 2x 縮放：綠色「2.0x」
  - 全螢幕：白色「1.0x」起
- **顏色變化：**
  - 調整時短暫變綠色
  - 穩定後恢復半透明黑

### 控制按鈕
- **顯示：** 滑鼠移到畫面時淡入
- **🔍 縮放按鈕：**
  - 未縮放：🔍
  - 已縮放：🔍⊖
- **⛶ 全螢幕按鈕：**
  - 未全螢幕：⛶
  - 已全螢幕：⛶⊖

### 滑鼠游標
- **標準：** 指標 (pointer)
- **可縮放：** 指標 + hover 效果
- **已縮放：** 移動 (move)
- **拖動中：** 抓取 (grabbing)

## 🔧 技術細節

### 縮放實現
- **標準縮放：** CSS transform scale(2)
- **滾輪縮放：** 動態 scale(1-5)
- **平移：** translate() 配合拖動事件

### 全螢幕 API
使用標準 Fullscreen API，支援：
- Chrome/Edge: `requestFullscreen()`
- Firefox: `mozRequestFullScreen()`
- Safari: `webkitRequestFullscreen()`
- IE11: `msRequestFullscreen()`

### 性能優化
- 使用 CSS transitions 流暢動畫
- 防抖處理滾輪事件
- 智慧的 transformOrigin 計算

## ⚠️ 注意事項

### 效能考量
- 縮放不會影響錄影品質
- 縮放僅影響顯示，不改變實際解析度
- 全螢幕模式不會增加 CPU 負載

### 使用限制
- 縮放和全螢幕不能同時使用
- 全螢幕模式下不支援固定 2x 縮放
- 改用滾輪縮放取得更好的控制

### 相容性提示
- 部分瀏覽器需要用戶互動才能進入全螢幕
- 自動進入全螢幕可能被瀏覽器阻擋
- 建議使用最新版本瀏覽器

## 🐛 故障排除

### 問題：點擊畫面沒有反應
**解決方案：**
1. 確認預覽已啟動（畫面有顯示）
2. 確認沒有在錄影中
3. 重新整理頁面

### 問題：全螢幕無法啟動
**可能原因：**
1. 瀏覽器安全限制
2. 需要用戶手動點擊

**解決方案：**
1. 確保手動點擊按鈕（不要用自動化腳本）
2. 檢查瀏覽器全螢幕權限設定

### 問題：滾輪縮放沒反應
**確認事項：**
1. 必須先進入全螢幕模式
2. 滑鼠必須在畫面上
3. 標準模式請使用 🔍 按鈕

### 問題：拖動不順暢
**優化建議：**
1. 確認顯卡驅動已更新
2. 關閉瀏覽器硬體加速
3. 降低預覽解析度

## 📊 效能建議

### 最佳化檢視設定
- **一般監控：** 標準視圖
- **對焦調整：** 2x 縮放
- **細節檢查：** 全螢幕 + 3x-4x 滾輪縮放
- **長時間錄製：** 標準視圖，定期檢查

### 資源使用
- CPU 使用率：< 5% (僅視圖變換)
- 記憶體：無額外占用
- GPU：使用硬體加速（transform）
